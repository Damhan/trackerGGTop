# trackerGGTop

A Web scraper written using selenium in python.

Takes the top 100 tft players from tracker.gg and writes them to a CSV.

# Install & Run

- Clone repo
- Pip install selenium
- Install the latest chromedriver from online.
- CD into directory.
- python3 main.py
